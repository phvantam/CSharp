# TuneVault Frontend

## Cài đặt dự án

Clone repository:

```bash
git clone https://github.com/phvantam/CSharp.git
```

Di chuyển vào thư mục frontend:

```bash
cd CSharp/tunevault-frontend
```

Cài đặt dependencies:

```bash
npm install
```

Chạy development server:

```bash
npm run dev
```

Sau khi chạy thành công, mở trình duyệt tại:

```txt
http://localhost:5173
```
